Exercício 1: Contar Vogais e Consoantes

Objetivo: Criar um objeto que conte quantas vogais e consoantes tem uma palavra.

const palavra = "Cachorro";
Resultado esperado: {3 "C", 2 "O" e 2 "R"}
Desafio extra: Tente modificar o código para ignorar maiúsculas e minúsculas!
const palavra = "Cachorro".toLowerCase();
let letras = {};
for (let i = 0; i < palavra.length; i++) {
if (letras[palavra[i]]) {
letras[palavra[i]]++;
} else {
letras[palavra[i]] = 1;
}
}
console.log(palavra);
console.log(letras);

Explicação
  • const palavra = "Cachorro".toLowerCase();
    ➜ Transforma a palavra "Cachorro" em "cachorro" (tudo minúsculo).
    letras = {};
    ➜ Cria um objeto vazio que vai guardar as letras e quantas vezes cada uma aparece.
    for (let i = 0; i < palavra.length; i++)
    ➜ Vai percorrer cada letra da palavra, uma por uma.
    Dentro do for, o código verifica:
    Se a letra já existe no objeto letras, soma +1 na contagem.
    Se ainda não existe, cria a chave com valor 1.
    console.log(palavra);
    ➜ Mostra a palavra "cachorro".
    console.log(letras);
    ➜ Mostra o resultado final.

    • Exercício 3: Calculadora IMC

    Objetivo: Crie um programa em JavaScript que imprima 3 níveis de IMC = Índice de Massa Corporal, seguindo a fórmula.

    const imc = peso / (altura * altura)
    Resultado esperado:
    <= 18.5 → Abaixo do peso
    18.5 – 24.9 → Peso normal
    >= 25 → Sobrepeso
    const peso = 55;
    const altura = 1.7;
    const imc = Number((peso / (altura * altura)).toFixed(2));
    console.log(imc);
    switch (true) {
    case imc < 18.5:
    console.log("Você está abaixo do peso!");
    break;
    case imc >= 18.5 && imc <= 24.9:
    console.log("Seu peso está normal!");
    break;
    case imc >= 25:
    console.log("Sobrepeso!");
    }
    Explicação
  • peso / (altura * altura) = 55 ÷ (1.7 × 1.7) = 19.03
    .toFixed(2) = arredonda para 2 casas decimais
    Number() = garante que seja um número
    switch normalmente compara valores exatos: switch(cor)
    Mas aqui queremos testar condições (maior que, menor que)
    switch (true) significa: "execute o primeiro case que for verdadeiro"
    Exemplo: "19.03 é menor que 18.5?" → NÃO?
    Ação: Pula para o próximo case.
